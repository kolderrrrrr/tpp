# Лабораторная работа 3: менеджер заметок WPF

WPF-приложение для создания, редактирования, удаления, поиска и сохранения текстовых заметок.

## Возможности

- добавление новой заметки через `TextBox`;
- отображение заметок в `ListBox`;
- загрузка выбранной заметки в поле редактирования;
- сохранение изменений выбранной заметки;
- удаление выбранной заметки;
- автоматическое сохранение заметок в файл `notes.txt` при закрытии приложения;
- загрузка сохраненных заметок при следующем запуске;
- поиск заметок по содержимому;
- изменение цвета фона поля заметки через контекстное меню.

## Запуск

1. Открыть `TppLabs.sln` в Visual Studio 2022.
2. Выбрать проект `NotesWPF` как запускаемый.
3. Нажать `F5`.

Файл `notes.txt` создается автоматически рядом с исполняемым файлом приложения и хранит заметки в текстовом JSON-формате.
