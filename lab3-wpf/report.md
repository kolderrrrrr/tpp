# Отчет по лабораторной работе 3

## Тема

Разработка интерактивного WPF-приложения для создания, редактирования, удаления и сохранения текстовых заметок.

## Цель работы

Научиться разрабатывать полноценное интерактивное приложение с использованием Windows Presentation Foundation, элементов управления `TextBox`, `ListBox`, кнопок, контекстного меню и файлового хранения данных.

## Краткое описание проекта

В ходе лабораторной работы было создано WPF-приложение `Заметки`. Оно позволяет пользователю вести список текстовых заметок: добавлять новые записи, выбирать существующие записи из списка, редактировать их, удалять и искать по содержимому.

Заметки автоматически сохраняются в файл `notes.txt` при закрытии приложения и загружаются при следующем запуске. Для корректного хранения многострочных заметок используется текстовый JSON-формат.

## Использованные технологии

В работе использовались:

- Microsoft Visual Studio;
- платформа .NET;
- технология Windows Presentation Foundation;
- язык разметки XAML;
- язык программирования C#;
- файловый ввод-вывод для сохранения заметок.

## Ход выполнения работы

Сначала был создан WPF-проект `NotesWPF`. В главном окне приложения была настроена сетка `Grid`, которая делит интерфейс на список заметок, поле поиска, область редактирования и панель кнопок.

Фрагмент разметки списка заметок:

```xml
<ListBox x:Name="NotesListBox"
         SelectionChanged="NotesListBox_SelectionChanged"
         Background="White"
         BorderBrush="#C7D0DB" />
```

Для ввода и редактирования заметки используется многострочный `TextBox`. При выборе элемента списка текст выбранной заметки загружается в поле редактирования.

Фрагмент обработчика выбора заметки:

```csharp
private void NotesListBox_SelectionChanged(object sender, SelectionChangedEventArgs e)
{
    if (_isSelectionChanging || NotesListBox.SelectedItem is not NoteListItem item)
    {
        return;
    }

    NoteTextBox.Text = _notes[item.Index];
    ShowStatus("Заметка загружена для редактирования.");
}
```

Кнопка `Добавить` добавляет текст из поля ввода в коллекцию заметок. Кнопка `Сохранить` обновляет выбранную заметку, а кнопка `Удалить` удаляет выбранный элемент из списка.

Фрагмент добавления заметки:

```csharp
private void AddButton_Click(object sender, RoutedEventArgs e)
{
    string note = NoteTextBox.Text.Trim();

    if (string.IsNullOrWhiteSpace(note))
    {
        ShowStatus("Введите текст заметки.");
        return;
    }

    _notes.Add(note);
    NoteTextBox.Clear();
    RefreshNotesList();
    ShowStatus("Заметка добавлена.");
}
```

Для поиска заметок используется отдельное поле `SearchTextBox`. При изменении текста поиска список обновляется и показывает только те заметки, которые содержат введенную строку.

## Сохранение данных

Для хранения заметок используется простой текстовый файл `notes.txt`. При запуске приложения заметки загружаются из файла, если он существует. При закрытии окна текущий список заметок сохраняется на диск. Данные записываются в JSON-формате, чтобы многострочные заметки не разбивались на несколько отдельных записей.

Фрагмент сохранения:

```csharp
private void SaveNotesToFile()
{
    string fileContent = JsonSerializer.Serialize(_notes, new JsonSerializerOptions
    {
        WriteIndented = true
    });

    File.WriteAllText(_storagePath, fileContent);
}
```

## Дополнительные возможности

В приложении реализованы дополнительные требования:

- поиск заметок по содержимому;
- изменение цвета фона поля ввода через контекстное меню.

Фрагмент контекстного меню:

```xml
<ContextMenu>

    <MenuItem Header="Светло-зеленый фон" Tag="#DFF5E1" Click="ColorMenuItem_Click" />
    <MenuItem Header="Светло-голубой фон" Tag="#E1F0FF" Click="ColorMenuItem_Click" />
</ContextMenu>
```

## Возникшие трудности

Основная сложность заключалась в том, чтобы совместить поиск заметок с редактированием выбранного элемента. Для этого был создан вспомогательный класс `NoteListItem`, который хранит не только текст заметки, но и ее исходный индекс в общей коллекции.

В результате выполнения лабораторной работы было создано интерактивное WPF-приложение для управления заметками. Были изучены элементы интерфейса `TextBox`, `ListBox`, `Button`, `ContextMenu`, обработка событий, фильтрация данных и сохранение информации в текстовый файл.
