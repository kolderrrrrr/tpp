# Лабораторная работа 8 - LocalizedNotepadWPF

WPF-приложение `Многоязычный блокнот` с динамическим переключением языка интерфейса.

## Возможности

- текстовое поле для редактирования документа;
- меню `Создать`, `Открыть`, `Сохранить`, `Закрыть`;
- переключение языка интерфейса без перезапуска;
- строки интерфейса вынесены в `Resources/Strings.ru.resx` и `Resources/Strings.en.resx`;
- открытие и сохранение файлов `*.txt`;
- подтверждение закрытия при несохраненных изменениях;
- обработка ошибок при работе с файлами;
- возможность добавить новый язык через дополнительный файл `Strings.<код>.resx`.

## Запуск

1. Открыть `TppLabs.sln` в Visual Studio 2022.
2. Выбрать проект `LocalizedNotepadWPF` стартовым.
3. Нажать `F5`.
