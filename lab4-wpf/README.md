# Лабораторная работа 4 - GuessNumberWPF

WPF-приложение с игрой `Угадай число`.

## Возможности

- генерация случайного числа от 1 до 100;
- проверка введенного числа;
- подсказки `Слишком большое` и `Слишком маленькое`;
- отображение количества попыток;
- проверка корректности ввода;
- перезапуск игры без закрытия приложения;
- обновление интерфейса через привязку данных и `INotifyPropertyChanged`.

## Запуск

1. Открыть `TppLabs.sln` в Visual Studio 2022.
2. Выбрать проект `GuessNumberWPF` стартовым.
3. Нажать `F5`.
