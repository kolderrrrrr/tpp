# Лабораторная работа 7 - CustomButtonTemplateWPF

WPF-приложение для демонстрации пользовательского шаблона кнопки через `ControlTemplate` и `Style`.

## Возможности

- собственный `ControlTemplate` для элемента `Button`;
- использование `Border`, `Grid` и `ContentPresenter` внутри шаблона;
- поддержка свойств `Background`, `Foreground`, `BorderBrush`, `Padding`;
- визуальные состояния `MouseOver`, `Pressed`, `Disabled`;
- плавное масштабирование кнопки при наведении;
- изменение цвета и прозрачности для разных состояний;
- применение одного стиля к нескольким кнопкам;
- демонстрация, что кнопки сохраняют обработку кликов.

## Запуск

1. Открыть `TppLabs.sln` в Visual Studio 2022.
2. Выбрать проект `CustomButtonTemplateWPF` стартовым.
3. Нажать `F5`.
