# Лабораторная работа 1: WPF Hello, World!

## Что нужно сделать

Создать простое WPF-приложение, которое выводит на экран сообщение `Hello, World!`.

## Как выполнить в Visual Studio

1. Открыть Visual Studio 2019 или 2022.
2. Выбрать `Создать новый проект`.
3. Найти шаблон `WPF Application` / `Приложение WPF`.
4. Назвать проект `HelloWorldWPF`.
5. Открыть файл `MainWindow.xaml`.
6. Заменить содержимое файла на код из `MainWindow.xaml` в этой папке.
7. Запустить проект клавишей `F5`.
8. Проверить, что в окне отображается текст `Hello, World!`.

## Что приложить к отчету

- цель работы;
- краткое описание WPF и XAML;
- фрагмент кода `MainWindow.xaml`;
- скриншот запущенного приложения;
- вывод о результате работы.

## Важное замечание

WPF запускается только на Windows, поэтому на macOS приложение обычно не получится собрать и проверить напрямую. Код из этой папки нужно вставить в проект Visual Studio на Windows.
