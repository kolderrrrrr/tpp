# Лабораторная работа 2: WPF-калькулятор

## Что нужно сделать

Создать графическое WPF-приложение `Калькулятор`, которое выполняет базовые арифметические операции:

- сложение;
- вычитание;
- умножение;
- деление;
- вычисление выражений с учетом приоритета операций.

Дополнительно реализованы десятичные дроби, скобки и история вычислений.

## Как выполнить в Visual Studio

1. Открыть Visual Studio 2019 или 2022.
2. Создать новый проект `WPF Application` / `Приложение WPF`.
3. Назвать проект `CalculatorWPF`.
4. Заменить содержимое файла `MainWindow.xaml` кодом из этой папки.
5. Заменить содержимое файла `MainWindow.xaml.cs` кодом из этой папки.
6. Запустить проект клавишей `F5`.
7. Проверить выражения:
   - `2+3*4` должно дать `14`;
   - `(2+3)*4` должно дать `20`;
   - `10/0` должно показать сообщение об ошибке.

## Что приложить к отчету

- цель работы;
- краткое описание проекта;
- основные функции калькулятора;
- фрагменты `MainWindow.xaml` и `MainWindow.xaml.cs`;
- скриншоты интерфейса и результата вычислений;
- вывод о результате работы.

## Важное замечание

WPF запускается только на Windows. На macOS код можно подготовить, но для сборки и проверки приложения обычно нужен Visual Studio на Windows.
