# Лабораторная работа 5 - NotepadWPF

WPF-приложение `Блокнот`, реализующее работу с текстовыми файлами через команды.

## Возможности

- создание нового документа;
- открытие текстового файла;
- сохранение текущего документа;
- сохранение нового документа через диалог выбора файла;
- команды меню и горячие клавиши `Ctrl+N`, `Ctrl+O`, `Ctrl+S`;
- обработка ошибок при открытии и сохранении файлов;
- отображение текущего имени файла и признака несохраненных изменений.

## Запуск

1. Открыть `TppLabs.sln` в Visual Studio 2022.
2. Выбрать проект `NotepadWPF` стартовым.
3. Нажать `F5`.
