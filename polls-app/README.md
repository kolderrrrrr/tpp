# PollsApp

Минимальная версия курсового проекта: веб-приложение для создания и проведения опросов и голосований с анализом результатов.

## Возможности

- регистрация и вход пользователя;
- создание опроса;
- добавление вопросов с вариантами ответа;
- публикация опроса по публичной ссылке;
- прохождение опубликованного опроса;
- защита от повторного голосования через cookie и учетную запись;
- просмотр результатов в виде счетчиков и простых диаграмм;
- REST API для базовых операций.

## Запуск

```bash
go run ./cmd/server
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:8080
```

Данные сохраняются в файл `data/store.json`.

## Стек

- Go;
- стандартная библиотека `net/http`;
- HTML, CSS;
- JSON-файл как простое локальное хранилище для минимальной версии.

## REST API

```http
POST /api/auth/register
POST /api/auth/login
GET  /api/polls
POST /api/polls
POST /api/polls/{id}/questions
POST /api/polls/{id}/publish
GET  /api/polls/{id}/results
```

## Перспективы развития

- перенос хранения данных на PostgreSQL или SQLite;
- JWT или полноценные серверные сессии;
- редактирование и удаление отдельных вопросов;
- экспорт результатов в CSV;
- Swagger-документация;
- тесты сервисного слоя.
