# Лабораторная работа 9 - MultiStepFormWPF

WPF-приложение с многоступенчатой формой и навигацией между страницами.

## Возможности

- главное окно с `Frame` для отображения страниц;
- три отдельные страницы `Page`;
- страница личных данных: имя, фамилия, дата рождения;
- страница контактных данных: email и телефон;
- страница адреса: город, улица, дом и квартира;
- кнопки `Назад`, `Далее`, `Отправить`;
- сохранение введенных данных между страницами через общий класс `FormData`;
- валидация обязательных полей, email и телефона;
- итоговый вывод всех данных через `MessageBox`.

## Запуск

1. Открыть `TppLabs.sln` в Visual Studio 2022.
2. Выбрать проект `MultiStepFormWPF` стартовым.
3. Нажать `F5`.
