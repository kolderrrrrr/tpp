# Лабораторная работа 6 - AnimatedCalculatorWPF

WPF-приложение `Анимированный калькулятор` с базовыми арифметическими операциями и анимационными эффектами.

## Возможности

- ввод чисел кнопками от 0 до 9;
- операции сложения, вычитания, умножения и деления;
- вычисление результата кнопкой `=`;
- очистка дисплея кнопкой `C`;
- последовательное выполнение операций;
- масштабирование кнопок при наведении;
- изменение цвета кнопки при нажатии;
- анимация появления результата на дисплее;
- анимация очистки дисплея.

## Запуск

1. Открыть `TppLabs.sln` в Visual Studio 2022.
2. Выбрать проект `AnimatedCalculatorWPF` стартовым.
3. Нажать `F5`.
